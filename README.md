# 📍 Directorio Comercial de Ancón

¡Bienvenido al repositorio del **Directorio Comercial de Ancón**! Este proyecto es una plataforma web diseñada para centralizar, organizar y dar visibilidad a los comercios, emprendimientos y negocios locales de la zona de Ancón. 

El objetivo principal es conectar a los residentes y visitantes con la oferta comercial de la comunidad, impulsando el crecimiento local a través de la tecnología.

---

## 🚀 Características del Proyecto
* **Directorio Centralizado:** Listado organizado por categorías (restaurantes, servicios, tiendas, etc.).
* **Fácil Navegación:** Interfaz limpia y responsive construida con HTML para un acceso rápido desde cualquier dispositivo.
* **Información Local de Valor:** Datos de contacto, ubicación y horarios de los comercios del área.

---
